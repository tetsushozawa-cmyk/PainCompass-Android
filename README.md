# Pain Record

Android Studioで開いて、そのままAndroidスマホに実行できる最小構成のアプリです。

## 動かし方

1. Android Studioで `/Users/shozawatetsu/AndroidStudioProjects/PainRecord001` フォルダを開きます。
2. Gradle Syncが終わるまで待ちます。
3. 画面上部の実行設定が `app` になっていることを確認します。
4. AndroidスマホをUSB接続するか、エミュレーターを起動します。
5. 実行ボタンを押します。

実行ボタンが出ない場合は、`File > Sync Project with Gradle Files` を押してから、上部の実行設定で `app` を選んでください。`app` フォルダだけではなく、必ずプロジェクト全体の `PainRecord001` フォルダを開きます。

## アプリでできること

- トップ画面に「線維筋痛症ダイアリー」と表示
- 「記録を入力する」ボタンから記録入力画面へ移動
- 「保存を見る」ボタンから保存済み記録一覧へ移動
- 「曼荼羅用データ」ボタンから過去7回分の痛み・動ける状態の番号を表示
- 曼荼羅用データ画面に6×6マップを表示
- 記録入力画面で現在日時、痛みの度合い、動ける状態、睡眠の状態、頭の働き、痛い時間帯、メモを保存
- 痛い時間帯はチェックボックスで複数選択
- 「次へ」ボタンから、天候・気温・気圧の変化・日常活動を入力できる次ページへ移動
- 次ページの保存ボタンで、前ページの1〜5項目と次ページの6〜9項目をまとめて保存
- 保存済み記録一覧で記録を確認・削除
- 入力画面、次ページ、保存一覧画面から「最初に戻る」でトップへ戻る
